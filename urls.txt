# NOTES FOR SCALING UP
# ========================================
# 1. Download full CubiCasa5k dataset (5000 images): 
#    wget https://github.com/CubiCasa/CubiCasa5k/archive/refs/heads/master.zip
#
# 2. Access FloorPlanCAD dataset (15,000+ CAD drawings):
#    https://floorplancad.github.io/
#
# 3. MLSTRUCT-FP dataset (954 floor plans):
#    https://github.com/MLSTRUCT/MLSTRUCT-FP
#
# 4. Download from Kaggle datasets:
#    - CubiCasa5k: https://www.kaggle.com/datasets/qmarva/cubicasa5k
#    - Architecture dataset: https://www.kaggle.com/datasets/wwymak/architecture-dataset
#
# 5. Generate custom site plans with CADMAPPER:
#    https://cadmapper.com/ (up to 1kmÂ² free)
#
# 6. Access urban planning DWG files:
#    - https://dwgmodels.com/urban_planing/
#    - https://www.archweb.com/en/cad-dwg/category/urban-planning/
#    - https://www.bibliocad.com/en/library/urban-design/

# ========================================
# PROCESSING TIPS
# ========================================
# - Reduce processing interval to 5-10 seconds for faster learning
# - Run multiple learner instances in parallel
# - Process high-quality datasets first (CubiCasa5k)
# - Include variety: residential, commercial, site plans
# - Mix styles: modern, traditional, CAD, hand-drawn
# - Include different regions for global patterns

# Total potential training images: 20,000+
# - CubiCasa5k: 5,000
# - FloorPlanCAD: 15,000+
# - MLSTRUCT-FP: 954
# - Plus hundreds from various sources above
